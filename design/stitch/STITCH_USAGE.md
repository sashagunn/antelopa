# Stitch Usage Notes

The files in /design/stitch are not full page definitions.

They are design references only.

There are:
- 5 mobile design references
- 5 desktop design references

These files should be used as a visual system reference for:
- typography
- spacing
- color palette
- card styles
- section layout
- image treatment
- buttons
- forms
- gallery styling
- mobile responsiveness
- desktop responsiveness

Important:
Do not treat each Stitch file as a separate finished page.

Instead:
Use them as design inspiration and UI reference when redesigning the existing site.

The current site structure, sections, and text content remain the source of truth.
Only the visual style should be updated to match the Stitch references.
